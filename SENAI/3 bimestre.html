Projeto de Site

Nesse trabalho, eu e meu grupo (Erico, Gabriel, Gustavo, João Pedro e Sirlan) criamos o conceito de um site com slides no Canva. Basicamente, nós pensamos em um site no qual teria foco exclusivamente em criar questões para estudar para vestibulares, diante disso, nós pensamos em um site chamado QuestLab. Eu e outros membros criamos o design, no qual tem uma página para especificar como vão ser feitas as questões, uma para realizar as questões e outra para ver as respostas. No final, nós conseguimos expressar bem a nossa ideia de como seria o site, e foi uma boa experiencia para entendermos como funciona o planejamento na construção de sites.


Tradutor executavel

Nessa atividade, eu precisei usar a linguagem Python para criar sozinho um tradutor na aula de A.D.S. Para isso, eu usei a Tkinter e a própria biblioteca do GoogleTranslator para criar o arquivo, no qual depois de programar, eu fiz uma série de comandos para transforma meu arquivo Python em um arquivo .exe (EXECUTAVEL). No fim, posso afirmar com certeza que essa atividade foi ótima para a minha experiencia no curso, e que vai ser bem útil para futuros projetos pessoais que eu possa tentar fazer.
